# 🌍 Voyageur – Travel & Restaurant Website

A modern, responsive travel and restaurant web platform built using HTML, CSS, and JavaScript.  
Voyageur delivers a luxury UI experience with interactive features for exploring destinations, dining options, and travel booking.

---

## ✨ Features

- 🏝️ Hero slider with auto-play, swipe, and keyboard navigation  
- 🔍 Live search, category filters, and sorting system  
- ❤️ Wishlist functionality for destinations  
- 🍽️ Restaurant listings with cuisine filters and badges  
- 🖼️ Image gallery with lightbox (keyboard + swipe support)  
- 🗺️ Interactive map using Leaflet.js with location markers  
- 📅 Booking form with validation and dynamic price preview  
- 🌙 Dark/Light mode toggle (saved in local storage)  
- 📱 Fully responsive design (mobile-first)  
- ⚡ Optimized performance (lazy loading, animations, debounce search)

---

## 🛠️ Tech Stack

- HTML5  
- CSS3 (Flexbox, Grid, Animations)  
- JavaScript (ES6+)  
- Leaflet.js (Maps API)
